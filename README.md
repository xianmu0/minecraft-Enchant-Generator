# minecraft-Enchant-Generator
minecraft物品附魔命令生成
Minecraft 附魔指令生成器
一个强大且易用的《我的世界》附魔指令生成工具，支持图形化界面操作，助你快速生成复杂的附魔 NBT 指令。本项目是一个为《我的世界》玩家和开发者设计的附魔指令生成器。你无需记忆复杂的 NBT 标签语法，只需通过简单的点选和输入，即可生成精确的 /give 或 /replaceitem 指令，用于为物品附加自定义附魔。
无论是生存模式的便利工具，还是创造模式地图制作的得力助手，这个生成器都能极大提升你的效率。直观的图形界面 (GUI): 告别手动输入 NBT 的烦恼，所有操作皆可通过界面完成。
在线使用推荐访问项目的 GitHub Pages 页面 

使用教程:
1.在界面左侧选择目标物品
2.在附魔列表中选择你想要的附魔并下拉选择等级
3.点击“复制指令”按钮
4.右侧文本框会自动生成完整指令，点击“复制”按钮即可使用。
